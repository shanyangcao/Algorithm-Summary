### 二叉树的迭代遍历

二叉树中递归能做的，栈也能做

#### 前序--栈

1. 压入根节点
2. 在栈不空的情况下，**栈**弹出了一个节点，**集合**记录此节
3. 分别对左右孩子进行处理栈操作
4. 返回集合中树的遍历结果

```
stack.push(root);
while (!stack.isEmpty()){
    TreeNode node = stack.pop();
    result.add(node.val);
    if (node.right != null){
        stack.push(node.right);
    }
    if (node.left != null){
        stack.push(node.left);
    }
}
return result;
```

#### 后序--栈

1. 压入根节点
2. 在栈不空的情况下，**栈**弹出了一个节点，**集合**记录此节点
3. 分别对左右孩子进行处理栈操作
4. **翻转**result
5. 返回集合中树的遍历结果

```
stack.push(root);
while (!stack.isEmpty()){
    TreeNode node = stack.pop();
    result.add(node.val);
    if (node.left != null){
        stack.push(node.left);
    }
    if (node.right != null){
        stack.push(node.right);
    }
}
Collections.reverse(result);
return result;
```

#### 中序--栈

1. 记当前root为cur，cur此时就相当于指针
2. **结束条件**，指针不为空且栈不为空的情况下
3. 如果指针不为空，则栈压入cur，指针指向左孩子
4. 如果指针为空，**栈**就弹出了最前面节点，**集合**记录此节点，指针指向右孩子

```
TreeNode cur = root;
while (cur != null || !stack.isEmpty()){
   if (cur != null){
       stack.push(cur);
       cur = cur.left;
   }else{
       cur = stack.pop();
       result.add(cur.val);
       cur = cur.right;
   }
}
return result;
```

### 翻转二叉树

翻转二叉树，其实就把每一个节点的左右孩子交换一下

也就是遍历每个节点，然后对节点的左右子树做翻转操作，那么选择什么遍历方式

#### 迭代

```
public TreeNode invertTree(TreeNode root) {
    if (root == NULL) return root;
    stack<TreeNode> st;
    st.push(root);
    while(!st.empty()) {
        TreeNode node = st.top();              // 中
        st.pop();
        swap(node.left, node.right);
        if(node.right) st.push(node.right);   // 右
        if(node.left) st.push(node.left);     // 左
    }
    return root;
}
```

#### 递归

```
public TreeNode invertTree(TreeNode root) {
    if (root == null) {
        return null;
    }

    //对左右子树
    invertTree(root.left);
    invertTree(root.right);
    //对左右孩子节点进行交换处理
    swapChildren(root);
    return root;
}

private void swapChildren(TreeNode root) {
    TreeNode tmp = root.left;
    root.left = root.right;
    root.right = tmp;
}
```

### 对称二叉树

#### 迭代

主要是后序遍历，左右中

收集孩子信息返回给上一级

#### 递归

1. 确定递归函数的参数返回值

首先看力扣给的函数

```
public boolean isSymmetric1(TreeNode root)
```

函数需要返回boolean，，并且比较是否是对称数，需要传入左右孩子节点，所以递归函数就是

```
bool compare(TreeNode left, TreeNode right)
```

1. 确定终止条件

就是比较两个节点数值相不相同，并且找好哪个条件的结果是进入递归逻辑

1. 确定单层递归的逻辑

比较左右节点的孩子值是否相等

- 比较二叉树外侧是否对称：传入的是左节点的左孩子，右节点的右孩子。
- 比较内侧是否对称，传入左节点的右孩子，右节点的左孩子。
- 如果左右都对称就返回true ，有一侧不对称就返回false 。

最终是

```
class Solution {
    public boolean isSymmetric1(TreeNode root) {
        return compare(root.left, root.right);
    }

    private boolean compare(TreeNode left, TreeNode right) {

        if (left == null && right != null) {
            return false;
        }
        if (left != null && right == null) {
            return false;
        }

        if (left == null && right == null) {
            return true;
        }
        if (left.val != right.val) {
            return false;
        }
        // 比较外侧
        boolean compareOutside = compare(left.left, right.right);
        // 比较内侧
        boolean compareInside = compare(left.right, right.left);
        return compareOutside && compareInside;
    }

}
```

再一次证明了只能是后序遍历

### 最大深度

```
class Solution {
    public int maxDepth(TreeNode root) {
        //递归
        if(root==null){return 0;}
        int rightDepth = maxDepth(root.right);//左
        int leftDepth = maxDepth(root.left);//右
        int Depth = 1+Math.max(rightDepth,leftDepth);//中
        return Depth;//后序遍历
    }
}
```
